 Clone this project:
 
  ```
  git clone git@github.com:douglasdrumz/vimfiles.git
  ln -s `pwd`/vimfiles ~/.vim
  ```
  
  or
  
  `git clone git@github.com:douglasdrumz/vimfiles.git ~/.vim`
 
 Then
 `git clone git@github.com:VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim`
 
 Open Vim  
 `:VundleInstall`
